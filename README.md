## About
Generates four different types of biomes - mountains, valleys, plains and deserts. Their boundaries are interpolated.

## Controls
- W,A,S,D,Left CTRL,Left Shift : Moving Front, Back, Left, Right, Up, Down
- P : Default Camera View
- M : Show Mesh
- K : Show Axes
